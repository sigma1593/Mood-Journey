# Mood Journey

## Descripción del Proyecto
Mood Journey es una aplicación móvil Android diseñada para ayudar a los usuarios a registrar sus emociones diarias mediante emojis y notas breves. La aplicación busca fomentar la reflexión personal y el bienestar emocional al permitir que los usuarios identifiquen patrones en su estado de ánimo a lo largo del tiempo.
El objetivo es ofrecer una herramienta simple e intuitiva para realizar seguimiento emocional y promover una mayor conciencia sobre las emociones y hábitos diarios.

## Problema que Resuelve
Muchas personas experimentan cambios emocionales relacionados con el trabajo, los estudios o situaciones personales, pero no cuentan con una herramienta sencilla para registrar y analizar sus emociones.
La falta de seguimiento emocional dificulta identificar patrones que pueden afectar el bienestar personal y la toma de decisiones.

## Objetivo General
Desarrollar una aplicación Android que permita registrar emociones diarias, almacenar notas personales y consultar un historial emocional para facilitar la reflexión y el autoconocimiento.
